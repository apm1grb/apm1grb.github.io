    Class DeviceManager<T>

    A DeviceManager<T> handles how a device is added, modified and removed in the UI. Depending upon the type of a device some devices appear as own nodes in the DeviceTree and others only as a table in the right pane. Therefore very distinct UI approaches are necessary to access and manage these devices.
